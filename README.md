# Creative Web Snippets

Welcome to **Creative Web Snippets** — a beginner-friendly open-source project created by **CSE Seniors of MITE** to help juniors learn Git, GitHub, and open-source collaboration! 

---

## What is This Project?

This repo collects **creative front-end code snippets** made by students — such as:
- Beautiful **HTML pages**
- Cool **CSS designs**
- Simple **JavaScript scripts**

Each student contributes their own file, learns how to make a Pull Request, and becomes an open-source contributor!
